Create an account on github.com (or another public project hosting repo).

Sign up for the free account on github (www.github.com), create an ssh-key and add it to your github account (http://help.github.com/mac-key-setup/). There's a good overview of how to use git and manage projects on the github help center (http://help.github.com/).


Create a simple Processing sketch and save it in a git repo.

Create an new repo named 201B-Processing-Sketches-(YourName). Then, inside your local repo, create and save a simple Processing sketch named Week1Day1, you can base it on an existing sketch if you are new to Processing. Add the new file(s) to be tracked by git, commit them to the local repository, and then push these commits to the remote github repository.

That is:
> git add (filenames) //this lets git start tracking the files
> git commit -a //this automatically adds or removes all tracked files to the local repository
> git push //this pushes the local changes to the remote repository (on github)
