Description
This course introduces fundamental concepts of computer science with a particular focus on computing with media data via object-oriented programming techniques. The goal of the course is to provide students with a practical foundation in programming and also to survey effective technologies useful for the creation of multimedia projects. To that end, we begin the course by introducing the Processing language, which provides a straightforward context in which to learn basic programming concepts, and then will migrate to the C++ and/or Java language by the end of the quarter. We also will, time permitting, take a brief look at some other languages along the way (C, Perl, GLSL, Python, JavaScript, Haskell). The main topics will include: object-oriented programming, data structures, algorithms, persistence, networking, concurrency, GUI programming, and graphics and audio libraries. In addition to frequent reading and quizzes, short programming exercises, collaborative experiments and a research presentation, students will be graded on the technical and creative aspects of a final project. The class will alternate between lectures and programming labs. There are no prerequisites; all students are required to bring a laptop to each class in order to participate in programming exercises.
